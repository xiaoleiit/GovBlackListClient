# 政务局黑名单信息对比

本目录用于归档政务局黑名单信息对比相关的联调材料，包含模拟服务端、客户端、样例文件和项目文档。正式客户端名称统一为“政务局黑名单信息对比客户端”。

## 目录说明

- `1、模拟服务端/`
  存放本地联调使用的 Spring Boot 模拟服务端。
- `2、对比客户端/`
  存放桌面客户端源码，当前实现位于 `alive-check-client/`。
- `3、生存待校验文件ZJPV/`
  存放输入样例文件，用于本地演示和联调。
- `4、生存已校验文件ZJPC/`
  存放输出样例文件，用于核对生成格式。
- `5、开发相关文档/`
  存放技术文档、接口文档及文档索引。

## 快速使用

1. 启动 `1、模拟服务端/unified-alive-auth`。
2. 进入 `2、对比客户端/alive-check-client/` 安装依赖并运行 `python3 main.py`。
3. 在客户端配置 `appId`、`appSecret`、接口地址和保存目录。
4. 选择 `3、生存待校验文件ZJPV/` 中的样例目录执行比对。
5. 生成结果可与 `4、生存已校验文件ZJPC/` 中的样例输出对照。
