# 开发相关文档索引

本目录用于集中管理政务局黑名单信息对比项目的文档材料，避免技术文档和接口依据散落在根目录。

## 子目录说明

- `技术文档/`
  存放项目内部技术说明书、架构说明和部署排障文档。
- `接口文档/`
  存放外部接口说明、第三方对接资料和原始接口依据。

## 文档生成

客户端技术文档由脚本生成：

```bash
cd "2、对比客户端/alive-check-client"
python3 -m pip install python-docx
python3 tools/generate_tech_doc.py
```

生成后的文档默认输出到 `5、开发相关文档/技术文档/`。
